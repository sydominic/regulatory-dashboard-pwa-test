Windows 작업 스케줄러 등록 방법

1. 시작 메뉴에서 "작업 스케줄러" 실행
2. 오른쪽 "작업 만들기" 클릭
3. 일반 탭
   - 이름: MFDS 로컬 수집기
   - 사용자의 로그온 여부와 관계없이 실행 또는 사용자가 로그온할 때만 실행 중 환경에 맞게 선택
4. 트리거 탭
   - 새로 만들기
   - 매일
   - 시작 시간 예: 08:30
5. 동작 탭
   - 새로 만들기
   - 프로그램/스크립트: mfds_collector\run_collect_mfds_scheduled.bat 전체 경로 선택
   - 시작 위치: 프로젝트 루트 폴더 또는 mfds_collector 폴더
6. 조건 탭
   - 노트북이면 "AC 전원일 때만" 조건은 필요에 따라 해제
7. 설정 탭
   - 예약된 시작 시간을 놓친 경우 가능한 빨리 실행 체크 권장

주의:
- PC가 꺼져 있으면 실행되지 않습니다.
- 회사망에서 Supabase 접속이 막히면 저장되지 않습니다.
- 로그는 mfds_collector\logs\scheduled_collect.log에 누적됩니다.
